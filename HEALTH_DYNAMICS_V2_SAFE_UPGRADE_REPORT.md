# Health Dynamics v2 Safe Upgrade Report

## What changed
- Replaced placeholder legal content with fuller Privacy Policy, Terms of Use, Medical Disclaimer, and Legal Notice text.
- Added local-first privacy language explaining that this static app stores user-entered data on the user's browser/device.
- Added warnings that the app does not provide medical advice, diagnosis, treatment, prescribing, or emergency guidance.
- Added a non-destructive live wellness calculator suite inside the existing calculator page.

## Live tools added
- BMI calculator
- Calories / macro estimate
- Water intake estimate
- Blood-pressure average tracker with CSV export
- Heart-rate zone estimate
- Sleep / mood / energy tracker with CSV export

## Safety notes
- This update does not remove or rewrite the original navigation system.
- The new files are add-ons: `health-dynamics-v2.css` and `health-dynamics-v2.js`.
- Legal text is a strong template, not legal advice. A qualified attorney should review before public release.
- This remains a local-first app. Data can be lost if browser data is cleared, so users should export backups.
