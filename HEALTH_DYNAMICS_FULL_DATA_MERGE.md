# Health Dynamics Full Data Merge

This build keeps the original peptide inventory web app intact and adds the complete Health Dynamics application as a full module at:

`health-dynamics/index.html`

The Health Dynamics iframe is lazy-loaded only after clicking the **Health Dynamics** sidebar tab.

Important for GitHub Pages:
- Upload the entire extracted folder contents.
- Do not upload only `index.html`.
- Make sure the `health-dynamics/` folder is present beside `index.html`.
- Make sure GitHub Pages is set to `main` and `/ (root)`.

This merge includes the full Health Dynamics files: legal pages, privacy, terms, disclaimer, cost analysis, dashboard, tabs, live calculators, medication, labs, journal, reports, and supporting CSS/JS/assets.
