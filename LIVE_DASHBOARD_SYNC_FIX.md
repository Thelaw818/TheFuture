# Live Dashboard Sync Fix

This build keeps the existing populated Health Dynamics data and makes the main dashboard cards update from saved local records.

Updated dashboard cards:
- Health Score: calculated from sleep, mood, energy, water, and steps.
- Sleep: latest saved sleep log or journal sleep value.
- Water: saved daily water and water goal.
- Steps: saved daily steps and step goal.
- Mood: latest saved mood value.
- Weight Progress: latest saved health or measurement weight.
- Latest Vitals: latest saved blood pressure and heart rate.
- Recent Activity chart: latest weight, blood pressure, and heart-rate records.
- Journal preview: latest saved journal note.

Added a dashboard Live Entry panel so the user can update weight/vitals, sleep/mood, water, and steps directly from the dashboard. Data remains local to the user's browser/device.
